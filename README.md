## Installation

### Prerequisites

Make sure you have the following installed:

- Node.js
- npm
- express
- nodemon
  

### Steps

1. Clone the repository:

   ```sh
   git clone https://github.com/ShijinA530/Movies.git
   cd your-repo

2. Open two terminals

   one for server and one for client to run

2. Run the server
   
   cd .\Server\
   nodemon app

4. Run the client
   
   cd .\Client\
   npm run dev
